1. Experiment phrase
`scrapy shell http://quotes.toscrape/random`

2. Generate simple scraper
`scrapy genspider quotes toscrape.com`

3. Run
`scrapy runspider quotes.py -o`
 
4. Save output to file 
`scrapy runspider quotes.py -o items.json`
